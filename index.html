Regles de calcul (CCNP IDCC 3097, Art. 34, contrat journalier)

Taux horaire = Minimum journalier / 8

Pour chaque journee :
1. Base = heures_effectives x taux ................. (toutes les heures)
2. Supp +100% = min(max(h_eff - 8, 0), 2) x taux .. (9e et 10e heure)
3. Supp +200% = max(h_eff - 10, 0) x taux x 2 ..... (a partir de la 11e)
4. Majo jour +50% = h_majo_jour x taux x 0.5 ....... (saisie manuelle)
5. Majo nuit +100% = h_majo_nuit x taux ............. (saisie manuelle)

Total jour = Base + Supp_100 + Supp_200 + Majo_jour + Majo_nuit
Cout employeur = Total_brut x (1 + taux_CS)

Verifie sur 4 bulletins reels (ITSI / 1618 Production). 104 metiers SNTPCT Art. 34 Prod. cine., deux grilles : accord mars 2024 et reevaluation juillet 2025.
